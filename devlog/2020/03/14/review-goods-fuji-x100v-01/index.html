2020년 03월 14일

Devlog Sass

x100v Black Unboxing


﻿스펙

  • 가격 : 1,699,000원
  • 종류 : 디지털카메라(똑딱이)
  • 26.1MP X-Trans CMOS4
  • 14bit RAW
  • 23mm F2 (환산 35mm) 렌즈 (6 군 8 매, 비구면 렌즈 2 매)
  • 4 스톱 ND 필터
  • 조리개 F2-16, 1 / 3EV 단계 9매 조리개 날개
  • 최단 촬영 거리 10cm
  • 컨버터 렌즈 대응
  • 디지털 텔레 컨버터 환산 50mm, 70mm
  • ISO160-12800(1 / 3EV 단계) 확장 ISO 80-51200 지원
  • 렌즈 셔터
  • 기계식 30 초 -1/4000 초, 전자식 30 초 -1 / 32000 초
  • 11fps 연사 JPEG 38매, RAW 17매, 8fps로 JPEG 76매, RAW 18매
  • 전자 셔터 연사 30fps, 1.25 배 크롭, JPEG 29ao, RAW 17매
  • 콘트라스트 -2EV 위상차 -5EV의 저휘도 AF
  • 시야율 95 %, 배율 0.52 배 OVF
  • 0.5 인치, 369 만 화소 유기 EL, 배율 0.66 배 EVF
  • 3 인치 3 : 2,162 만 화소
  • DCI4K 29.97p (10 분), 4K 29.97p ( 10 분) 전체 HD 59.97p (15 분)
  • 필름 시뮬레이션은 17 종
  • 배터리 NP-W126S, 배터리 수명 EVF 350 장, OVF 420 장
  • 크기 74.8 x 128.0 x 53.3mm
  • 무게 478g (배터리 및 SD 카드 포함), 428g (배터리 및 메모리 카드 제외)

기본 구성품

  • x100v 본체
  • 배터리 2개 (출시 이벤트 1+1)
  • c type 케이블
  • 어깨 스트랩

  • 설명서

                  참고로 기간내에 정품 등록하면 케이스 및 AS 1년 연장 가능.
                
              그리고 충격적인건 충전기가 없습니다.
                
              따로 구매 하시거나, c type 케이블로 충전하셔야 합니다.

사은품

  • “즐거운카메라” 에서 구입
  • ﻿카메라 청소도구
  • 액정 보호 필름
  • 바디 보호 필름
  • 64기가 메모리

  • 충전기

                    솔직히 충전기까지 사은품 준거는 깜짝 놀랐어요.
                  
                이걸 돈주고 사야하나 했는데... 사은품으로 똬!!
                  
                참고로 사은품은 사은품 입니다.
                  
                공식 홈페이지 안줍니다.
                  
                너무 사은품의 퀄리티는 논하지는 맙시다. 주는게 어디여



                x100 시리즈를 잘 모르시는 분들이 있어서 말씀드리지만
                
                정확히 디지털 카메라(똑딱이) 입니다.
                
                렌즈를 교체 하거나 할 수 있는, dslr, 또는 미러리스가 아닙니다.
                
                다만 박혀있는 렌즈에 어댑터링과 컨버터 렌즈를 따로 구매해서
                
                추가로 이어 붙이는 형식으로
                
                광각, 망원(최대 50mm) 으로 사용할 수 있습니다.



                        lcd 화면을 최대로 뺀 모습니다.
                        
                        돌아가거나 하지는 않습니다.
