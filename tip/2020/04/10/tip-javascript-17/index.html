jQuery(제이쿼리) 선택자에 변수 넣기

  • 직접 선택자를 정의해서 사용하지 않고, 이벤트로 특정 값을 가져와서 선택자로 사용할 경우 사용합니다.

index로 선택하기

let test = 'selectTest';
$('[index='+ test + ']');


id로 선택하기

let test = 'selectTest';
$('#'+ test );


class로 선택하기

let test = 'selectTest';
$('.'+ test );
