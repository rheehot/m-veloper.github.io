How to remove leading zeros from a number in JavaScript?



  1. 자바스크립트
let number = "0909123";
let formatNum = number.replace(/(^0+)/, "");
// 909123로 치환됨
console.log(formatNum);
  1. 자바
String testNum = "0909123";
String checkNum = testNum.replaceAll("^0+", "");
