MySQL REPLACE (콤마제거, 치환하기)

How to Using SELECT REPLACE with MySQL



  • SELECT문 사용시 컬럼에 콤마(,) 또는 치환해야할 문자가 있을 때 콤마를 제거하거나 문자를 치환해서 출력 하기 위해 사용

SELECT 시 그냥 REPLACE 만을 사용해서 콤마를 제거하는 경우

﻿SELECT ﻿REPLACE(컬럼, '컬럼 안에 바꿔야할 문자','바꿔서 출력될 문자') FROM 테이블 

﻿﻿SELECT ﻿REPLACE(price, ',','') FROM FOODS
  • 푸드 테이블의 price컬럼에서 콤마를 제거해서 출력하라



SELECT 시 REPLACE를사용해서 콤마를 제거 후 SUM 을 하는 경우

﻿SELECT SUM(REPLACE(price, ',','')) AS totalPrice FROM FOODS
  • 푸드 테이블의 price컬럼에서 콤마를 제거하고 price 컬럼을 모두 더해서 tatalPrice라는 이름으로 출력하라
